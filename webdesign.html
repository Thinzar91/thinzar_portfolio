Maison Belles fleurs

制作概要：高級フラワーブランド「ベル・フルール」の新レーベル『Maison Belles Fleurs』の公式Webサイトを制作しました。ブランドの世界観をWeb上で表現し、サービス内容や事例をわかりやすく伝えることを目的としたサイトです。
目的・ターゲット：空間装飾やフラワーデザインに関心のあるハイエンドな顧客層（企業・ホテル・富裕層個人）に向けて、「上質」「唯一無二」「オートクチュール」といったブランドコンセプトを視覚的・体験的に訴求することを目指しました。
工夫・ポイント：
• フラワーデザインの美しさを最大限に活かすため、大きなビジュアルと余白を重視したレイアウト。
• ブランドの高級感を引き立てるために、落ち着いたカラーと洗練されたタイポグラフィを使用。
• 更新性を考慮し、CMS（WordPress）を用いた構築で運用面にも配慮。
• スマートフォン対応のレスポンシブデザインを導入し、どのデバイスでも美しく閲覧できる設計に。
担当範囲：ワイヤーフレーム設計、Webデザイン、コーディング、CMS実装まで一貫して担当。
制作期間：約1ヶ月

使用ツール：Atom / Figma（ワイヤーフレーム） / HTML / CSS / JavaScript / Photoshop / Illustrator

本多屋 | 和食処

制作概要：Webデザインスクールの課題として、架空のカフェを想定したWebサイトを制作しました。トップページのデザインとコーディングを通じて、構成力・配色・レスポンシブ対応の基礎を学びました。
目的・ターゲット：「落ち着いた空間でコーヒーを楽しむ30代〜40代の大人世代」をターゲットに、店舗の雰囲気やメニューを魅力的に伝えることを意識しました。訪問者が視覚的に心地よく、操作しやすい設計を目指しています。
工夫・ポイント：
• 暖色系×余白を活かし、落ち着いた雰囲気を演出。
• メニューや店舗紹介はグリッドレイアウトで視認性を確保。
• ホバーアニメーションや画像切り替えなどの動きを加え、動的な印象に。
• モバイル表示でも読みやすいよう、レスポンシブデザインを実装。
担当範囲：デザイン設計からコーディング・公開まで全て担当。
制作期間：約1週間

使用ツール：Figma（ワイヤーフレーム） / HTML / CSS / JavaScript / GitHub Pages

Le Café des Chats | ル・カフェ・デ・シャ

制作概要：Webデザインスクールの課題として、架空のコンサルティング企業を想定したコーポレートサイトを制作しました。企業サイトに求められる信頼感や視認性を意識し、情報設計から実装までを一貫して行いました。
目的・ターゲット：ビジネス関係者や新規取引先など、信頼性を重視する訪問者を想定し、堅実かつ洗練された印象を与えるデザインを目指しました。特に読みやすさ・導線の明確さを重視しています。
工夫・ポイント：
• シンプルで見やすいグリッドレイアウトを採用。
• 色数を抑えた配色で落ち着きと信頼感を演出。
• ヘッダー・フッター・ナビゲーションの設計により、ユーザーが迷わず情報にたどり着けるよう配慮。
• HTML/CSS設計の基本構造を意識したマークアップと、レスポンシブ対応によるモバイル最適化。
担当範囲：デザイン設計からコーディング・公開まで全て担当。
制作期間：約1週間

使用ツール：Figma（ワイヤーフレーム） / HTML / CSS / JavaScript / GitHub Pages
